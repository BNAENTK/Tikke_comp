# Obsidian 자동 저장 정책

## 목표

`tikke` 관련 중요한 결정과 작업 요약만 Obsidian에 남긴다.

## 저장 대상

- `tikke` 관련 주요 결정.
- 배포 전후 요약.
- Claude Code에 전달할 작업 프롬프트.
- 아키텍처/운영 규칙 변경.
- 반복해서 쓸 체크리스트.

## 저장 제외

- 전체 대화 원문.
- 소스코드 원문.
- `.env` 내용.
- token, API key, secret, password, private key 값.
- 임시 디버그 로그 전체.
- 일회성 잡담.

## 저장 위치

- 중요 결정: `20_Projects/Tikke/Tikke 운영 원칙.md`
- 작업 요약: `20_Projects/Tikke/Tikke 작업 기록.md`
- Claude Code 프롬프트: `20_Projects/Tikke/Tikke Claude Code 협업.md`
- 자동화 기록: `30_Automation/Obsidian 자동 저장 정책.md`

## 원칙

- 저장 전 민감정보와 소스코드 원문을 제거한다.
- 요약 중심으로 저장한다.
- 사용자가 명시적으로 원문 저장을 요청해도 secret은 저장하지 않는다.
