# Tikke 운영 원칙

## 기본 원칙

- `tikke`는 TikTok LIVE 방송인용 툴킷 모노레포다.
- 새 기능은 별도 프로젝트가 아니라 `tikke` 모노레포 안에 추가한다.
- 배포는 사용자가 명시적으로 `배포해`라고 요청할 때만 진행한다.
- 버전 번호는 절대 내려가지 않는다.
- 화면에 보이는 한국어 UI 설명은 자연스러운 한국어로 작성한다.

## 안전 원칙

- 소스코드 원문을 Obsidian이나 문서화 repo에 복사하지 않는다.
- `.env`, token, API key, secret, password, private key 값은 저장하지 않는다.
- 정리본에는 구조, 기능, 운영 흐름, 체크리스트만 남긴다.
