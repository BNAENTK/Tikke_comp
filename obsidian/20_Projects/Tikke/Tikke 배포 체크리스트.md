# Tikke 배포 체크리스트

## 배포 전

- 사용자가 `배포해`라고 명시했는지 확인한다.
- 현재 최고 버전을 확인한다.
- 새 버전은 기존 버전보다 높게 잡는다.
- 변경 범위를 확인한다.
- 테스트 또는 빌드 검증 결과를 확인한다.

## 배포 범위

- Worker 배포는 명시 요청 시 진행한다.
- Pages 배포는 사용자가 `Pages 배포해`라고 별도 명시할 때만 진행한다.
- 자동 배포 금지.

## 배포 후 보고

- 버전.
- 변경 내용.
- 배포 대상.
