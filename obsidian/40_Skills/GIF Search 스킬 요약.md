# GIF Search 스킬 요약

## 용도
- Tenor API로 GIF를 검색하고 URL을 뽑거나 다운로드하는 스킬.

## 핵심 포인트
- `TENOR_API_KEY`가 필요하다.
- `curl + jq` 기반으로 가볍게 동작한다.
- full gif뿐 아니라 tinygif/mp4 같은 경량 포맷도 쓸 수 있다.

## 자주 쓰는 흐름
1. 키워드 검색
2. URL/메타데이터 확인
3. 바로 링크로 쓰거나 다운로드

## 주의사항
- API 키 없으면 바로 막힌다.
- 채팅/대시보드에선 tinygif 쪽이 더 실용적일 수 있다.
- 쿼리 URL 인코딩을 신경써야 한다.
