# Obsidian 스킬 요약

## 용도
- Obsidian vault 안의 노트를 읽고, 검색하고, 만들고, 수정할 때 쓰는 스킬.
- 대시보드에서 Obsidian을 흉내내기보다 실제 Obsidian 앱과 볼트를 직접 연결하는 방향에 적합하다.

## 핵심 포인트
- 이 사용자 볼트 경로는 `C:/Users/zkfma/Documents/BNAK_Knowledge`다.
- 기본 폴더 구조는 `00_Inbox`, `10_Wiki`, `20_Projects`, `30_Automation`, `40_Skills`, `90_Archive`다.
- `tikke` 관련 중요 작업은 `20_Projects/Tikke/Tikke 작업 기록.md`에 자동 저장하는 운영 규칙이 있다.

## 자주 쓰는 흐름
- 노트 읽기: 특정 md 파일 읽기
- 노트 검색: 파일명/본문 기준 검색
- 노트 생성: 필요한 폴더에 새 md 작성
- 노트 보강: 기존 노트의 특정 섹션에 append 또는 patch
- 실제 앱 열기: `obsidian://open?...` 링크 사용

## 주의사항
- 소스코드 원문, 비밀값, 긴 로그는 저장하지 않는다.
- 볼트 경로를 먼저 확정한 뒤 파일 도구를 사용한다.
- 대시보드 UI를 만들 때는 편집기보다 뷰어/노트 리스트 중심이 더 적합하다.
