# Architecture Diagram 스킬 요약

## 용도
- 다크 테마 기술 아키텍처/클라우드/서비스 다이어그램을 standalone HTML+SVG로 생성하는 스킬.

## 핵심 포인트
- 외부 렌더러 없이 단일 HTML 파일로 끝난다.
- 프론트/백엔드/DB/클라우드/보안 요소별 색 의미가 정해져 있다.
- 다크 그리드 기반의 기술적 느낌이 강하다.

## 자주 쓰는 흐름
1. 시스템 구성 요소/연결 파악
2. SVG 레이아웃 설계
3. HTML 파일 생성
4. 브라우저에서 시각 확인

## 주의사항
- 범례는 boundary 박스 밖에 둬야 한다.
- 화살표는 박스 뒤에 그려야 시각적으로 깔끔하다.
- 교육용/화이트보드 감성에는 Excalidraw 쪽이 더 잘 맞을 수 있다.
